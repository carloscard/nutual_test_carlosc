Se requiere tener instalado python para ejecutar el programa

Instalamos las librerías correspondientes que se encuentran en el archivo requirements.txt:

-pip install -r requirements.txt

Una vez instaladas las dependencias procedemos a correr el programa en local con el siguiente comando:

-python manage.py runserver

Una vez el servidor esté corriendo, obtendremos 3 vistas en total; 2 APIS y la ventana principal donde explotamos la información de las APIS:

-vista principal: http://127.0.0.1:8000/pisos/

-avm api: http://127.0.0.1:8000/api-avm/avm-api/

-avg m2 api: http://127.0.0.1:8000/api-avg/avg-m2-api/
